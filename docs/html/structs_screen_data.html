Detailed Description

Structure containing data to be passed to the taskActivatePumps.

This structure holds the text lines to be displayed on the LCD screen along with cursor position and blink control.

Parameters
linesArray of 4 strings, each up to 20 characters, representing the LCD lines.
lcdCursorXCursor X position (0-19).
lcdCursorYCursor Y position (0-3).
lcdCursorBlinkCursor blink flag (1 = blink enabled, 0 = disabled).
