Detailed Description

Custom data types for controlling application context in the DrinkCreator6000 project.

This module defines shared data structures and enumerations used to control the application state, including:

  • Menu and submenu numbers
  • LCD processing task input data
  • Information about the drinks
  • System error signature structure

These types are exchanged between FreeRTOS tasks and helper functions to maintain a consistent application context.

Note
Designed for use in a FreeRTOS environment but can also be reused in non-RTOS builds.
All types are project-specific and not tied to AVR hardware registers or peripherals.
