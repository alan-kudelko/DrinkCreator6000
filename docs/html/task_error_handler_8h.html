Detailed Description

Task responsible for stack overflow diagnostics and system error handling.

This task monitors guard zones to detect stack overflows in the project. Upon detecting an overflow, it records the event time and error cause in EEPROM, displays error information on the LCD and serial port, and restarts the system after a predefined timeout.
