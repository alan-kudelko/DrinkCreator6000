Detailed Description

Low-level UART driver for AVR (ATmega2560) with TX/RX buffering.

This module provides blocking and non-blocking UART routines with transmit and receive ring buffers. It is hardware-level and independent of any RTOS.

Two system-level ISRs (USART_UDRE_vect and USART_RX_vect) are defined internally in uart.c.

Note
TX and RX buffer sizes are configurable via macros.
Use uart_puts_P_* variants for strings stored in flash memory (PROGMEM).
This driver uses UART0 (USART0) peripheral on the ATmega2560, which has four UART interfaces.
