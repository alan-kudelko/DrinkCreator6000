Detailed Description

Blocking I2C (TWI) Master operations for AVR (ATmega2560) without TX buffer.

This header declares functions for performing I2C transactions in a blocking manner. All operations wait until the TWI hardware has completed the byte transfer before returning.

Note
These functions do not use a TX buffer and therefore occupy the CPU during transmission.
Suitable for simple or low-frequency I2C operations where non-blocking behavior is not required.
